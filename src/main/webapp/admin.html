Администрирование

Назначить время старта

Назначить время старта турнира по Московскому времени.
Запись 2012.12.31 19:59 означает декабрь 31-ое число, игры начнутся в 19:59 по Москве, регистрация откроется на несколько часов раньше (сейчас 6).

Назначить игру

Введите имена игроков в виде "Вася/Фрося".

Зарегистрировать игрока

Команда работает только в фазе регистрации, т.е. тогда же когда работает и обычный !register в общаке заграма.
НЕ использовать по личным просьбам! Это только для тестов или в случае каких-то сбоев.

Выставить статус турнира

Это то что слева-снизу около меню.

Назначить победителя

Введите имена игроков в виде "победитель/проигравший". Например, "Вася Новиков/Фрося Марусевна". Игрок "Вася Новиков" будет назначен выигравшим на во встрече. На заграме, при этом, игра эта закончена не будет.
